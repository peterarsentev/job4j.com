It is a short-crash course about programming in Java. This course consists from 14 lessons. Throughout the course, you will develop full app from scratch to the full project. You will start from simple tasks and go deeper throughout the next lessons.

Topics: Java 8, Gradle, Git, TestNG, SQL, Servlet, JSP, Hibernate, Spring (Boot, Data, Security), JAXB, SOAP, REST.

Each lesson has the three sections:
  • Theoretical material.
  • Practice.
  • Exercises.
