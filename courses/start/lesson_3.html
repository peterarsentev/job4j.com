General schema.


In our real life, we use boxes for storing, transporting or sorting all of our things.
In programming, we use variables to store, transport and sort things. We can do the same things like in real life with variables: storing, transporting, sorting.
Let’s look at a schema, which shows how to declare and initialize variables. 

[type] [name] [=] [value];

, where
type – It is a type of variable, like you use a jar for keeping liquid and you can’t use a paper box for it. A special type of container
name – it is identification of out variable, like you are human and you have a name. All variables must have names.
= - it is a symbol, which is used for assigning the value to our variables.
value – it is our data, information.
Java is stronged type program language. It means that you must declare variable with particular type and store the same value with such type.
Now here is a simple: 

public class Main {
    public static void main(String[] args) {
        int age = 25;
        char sex = 'm';
        boolean paid = false;
        System.out.println("Age : " + age);
        System.out.println("Sex : " + sex);
        System.out.println("Paid : " + paid);
    }
}

Here I created three variables: age, sex, paid. Each variables have a type.
A variable can’t keep the value with other types, which it was created.
The follow code is wrong: int age = ‘m’;
Because a variable age has int type, and we can keep only numbers in this variable.
primitive and reference
Java has two groups of variables types: primitive and reference.
In this lesson I am going to talk only about primitive types.

Java has 8 primitive types.
We can split it to 4 groups.
  • boolean, the type whose values are either true or false
  • char, the character type whose values are 16-bit Unicode characters
  • the integral types: byte, short, int, long
  • the floating-point types: float, double

Arithmetical operators.


We can use all arithmetical operators with primitive types.
Addition. 

int first = 10;
int second = 15;
int result = first + second;
System.out.println("Result : " + result);

Here you can see how we calculate the value and assign in directly to new variables 

int result = first + second;

Subtraction. 

int first = 10;
int second = 15;
int result = first - second;
System.out.println("Result : " + result);
Divide 

int first = 4;
int second = 2;
int result = first / second;
System.out.println("Result : " + result);

Multiplication. 

int first = 10;
int second = 15;
int result = first * second;
System.out.println("Result : " + result);

Reassign the values.


One interesting thing about variables is reassigning. We don’t have such operation in mathematic.

Let’s consider that I need to increment the value on one.

I can do in Java like this. 
int count = 0;
count = count + 1;
System.out.println("Count : " + count);


count = count + 1;
Java reads the value from count variable. Then it adds an one. Then it writes the value to variables count.

Java has simplified form of such expression: 
count += 10;

it is the same like count = count + 10;

If we talk only increment or decrement on one, you can use such simplified form. 
count++; count--;

Local-Variable Type Inference


Java 10 introduce a new simplified form of declare variable type.

Let’s consider the follow code; 
var price = 10.99;
System.out.println("Price : " + price);

Java extracts the type variable from value.

You can’t use such form if you don’t assign value.

Wrong! 
var price;
price = 10.99;
